# Frontend Assignment

Tech: TypeScript, React, Redux Toolkit, React Query, TanStack Router, Tailwind, ESLint + Biome, pnpm.

## Run

```bash
pnpm install
pnpm dev
```

## Scripts

- dev: start Vite dev server
- build: type-check then build
- preview: preview production build
- type-check: TypeScript no-emit check
- lint: ESLint
- format: Biome format
- lint:biome: Biome check
